# Retail_data_analysis_python_sql

**Python Data Cleaning + SQL Business Insights**  

This project focuses on transforming raw, unstructured data into a clean and analysis-ready dataset using **Python**, followed by solving real-world **business problems using SQL queries**.

---

## 📌 Project Overview  

Raw datasets often contain **missing values, inconsistent formats, and poorly structured columns**, which reduce data reliability and impact decision-making.  

In this project:  

- Cleaned and preprocessed the dataset using **Python (Pandas)**  
- Standardized column names and handled missing values  
- Performed advanced **business analysis using SQL**  
- Generated actionable insights for business growth  

---

## 🧰 Tech Stack  

- **Python (Pandas, NumPy)** – Data cleaning & preprocessing  
- **SQL** – Business analysis & querying  
- **Jupyter Notebook** – Development environment  
- **CSV Dataset** – Raw data source  

---

## ❗ Business Problem  

Organizations face challenges such as:  

- Poor data quality (null values, inconsistent column formats)  
- Lack of structured insights from raw data  
- Difficulty in tracking **sales trends, product performance, and growth metrics**  

---

## 🎯 Objective  

- Clean and prepare raw data for analysis  
- Ensure **data consistency and usability**  
- Solve real-world business problems using SQL  
- Extract insights to support **data-driven decision-making**  

---

## 🧹 Data Cleaning Steps (Python)  

- Handled **missing/null values**  
- Renamed columns for **consistency and readability**  
- Standardized data formats  
- Removed inconsistencies and ensured data quality  
- Prepared dataset for SQL-based analysis  

---

## 🧠 Business Questions Solved (SQL)  

- Identified the **sub-category with the highest profit growth** from **2022 to 2023**  
- Determined the **month with highest sales for each product category**  
- Performed **month-over-month (MoM) sales growth analysis** comparing **2022 vs 2023**  
- Extracted **top 5 highest-selling products in each region**  
- Identified the **top 10 revenue-generating products overall**  

---

## 📊 Key Insights  

- Certain sub-categories showed **significant year-over-year profit growth**, indicating strong market demand  
- Sales exhibited **seasonal trends**, with specific months consistently outperforming others  
- Month-over-month analysis revealed **growth patterns and fluctuations** across years  
- Regional analysis highlighted **top-performing products by geography**  
- A small set of products contributed disproportionately to overall revenue (**Pareto effect**)
